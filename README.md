# Lumen Veritatis — Scripture Assistant

> A scripture-focused AI chatbot that answers theological questions with RAG-grounded Bible references, remembers your conversation, and generates biblical artwork on request.

---

## Architecture

```
Browser  →  FastAPI (server.py)  →  LangGraph Agent (chatbot.py)
                                         ↓              ↓
                                     FAISS RAG     SD-Turbo
                                   (Bible verses)  (Images)
                                         ↓
                                     PostgreSQL
                                   (Conversation Memory)
```

The user sends a message from the browser. FastAPI passes it to the LangGraph agent, which decides whether to look up a Bible verse (RAG), generate an image, or answer directly. The full conversation is saved to PostgreSQL so the bot remembers context across the session.

---

## How It Works

### `chatbot.py` — The Agent

Built with **LangGraph** and **qwen3:8b via Ollama** (hosted on Google Colab, tunnelled via ngrok).

**`rag_tool`** — Searches a FAISS vector store of KJV Bible verses using `all-MiniLM-L6-v2` embeddings. Returns the top-5 matched verses so the LLM can cite scripture accurately without hallucinating.

**`image_generation_tool`** — Calls a `stabilityai/sd-turbo` server on Google Colab. The generated image is returned as base64 and displayed inline in the chat.

**Short-term memory** — LangGraph's `PostgresSaver` saves the full message history to PostgreSQL after every turn, keyed by `thread_id`. The bot remembers everything said earlier in the session — follow-up questions, prior topics, and previous image requests all stay in context.

### `server.py` — The Backend

**`POST /chat`** — accepts `{ message, thread_id? }`, runs the agent, and returns `{ text, thread_id, image_url?, grounded, hallucination_warning }`.

**`GET /`** — serves `index.html` directly, so no separate web server is needed.

### `index.html` — The Frontend

Single-page chat UI with Lumen Veritatis branding. Sends messages via `fetch`, renders scripture references as gold-styled links, and displays generated images inline.

## Dataset

Bible verses are sourced from the **Cleaned KJV Bible for LLMs** dataset:

🔗 https://huggingface.co/datasets/r-karra/Cleaned_KJV_Bible_for_LLMs

Verses are embedded with `all-MiniLM-L6-v2` at startup and stored in a FAISS index for fast similarity search.

## Setup

### Prerequisites

- Python 3.11+ with the `lang-graph` conda environment
- PostgreSQL running locally on `localhost:5432`
- `qwen3:8b` served via Ollama on Google Colab with an ngrok tunnel
- `sd-turbo` inference server on Google Colab with an ngrok tunnel

### Environment Variables

Create a `.env` file in the project root:

```env
DB_URL="postgresql://postgres:<password>@localhost:5432/"
IMAGE_GEN_URL="https://<your-ngrok-url>.ngrok-free.app/generate"
GOOGLE_API_KEY="<your-key>"
```

### Install & Run

```bash
conda activate lang-graph
pip install -r requirments.txt
uvicorn server:app --reload
```

Then open **http://127.0.0.1:8000** in your browser.

## Test Cases

### ✅ General theology question

**Input:** `Who was the apostle Paul?`

**Expected:** A detailed answer covering Paul's background as Saul of Tarsus, his conversion on the road to Damascus (Acts 9:1–19), his authorship of 14 New Testament epistles, and his missionary journeys — with scripture references hyperlinked in gold.

---

### ✅ Topic-based scripture lookup

**Input:** `What is the meaning of grace in Christianity?`

**Expected:** A structured breakdown with headings (Grace as Unmerited Favor, Grace as Salvation, Grace and the Cross, Grace and Daily Life), each backed by a verse retrieved via `rag_tool` (Romans 5:20, Ephesians 2:8–9, Romans 3:24, Titus 2:11–12).

---

### ✅ Hallucination prevention

**Input:** `Quote Genesis 52:10.`

**Expected:** `I could not verify this reference. The book of Genesis does not have a chapter 52. The longest chapter in Genesis is chapter 50.`

The bot refuses to invent a non-existent verse rather than generating plausible-sounding but false scripture.

---

### ✅ Image generation

**Input:** `Generate an image of Noah's Ark`

**Expected:** The bot calls `image_generation_tool`, the image is generated by `sd-turbo` on Colab, and displayed inline in the chat bubble.
